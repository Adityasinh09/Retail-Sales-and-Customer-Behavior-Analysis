# 🛍️ Retail Sales and Customer Behavior Analysis

## 📌 Project Overview

This project focuses on analyzing retail sales data to understand customer behavior, product performance, and revenue trends. The goal is to extract meaningful insights that can help businesses make data-driven decisions.

---

## 🎯 Objectives

* Analyze monthly sales trends
* Identify top-performing products
* Find the most profitable customer segments
* Understand purchasing behavior based on age and gender
* Explore quantity and pricing patterns

---

## 📊 Dataset

The dataset used in this project contains retail transaction data, including:

* Transaction ID
* Customer demographics (Age, Gender)
* Product Category
* Quantity purchased
* Price per unit
* Total purchase amount

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Aggregation and grouping
* Data visualization using charts and graphs

---

## 📈 Key Insights

* 📌 **Sales Trend:** Monthly sales show consistent patterns with certain peak periods indicating higher customer activity.

* 📌 **Top Product Categories:** Some product categories contribute significantly more to total revenue compared to others.

* 📌 **Customer Behavior:** Customers between the age group of 30–50 tend to spend more on average.

* 📌 **Purchase Patterns:** Most transactions include 1–3 items, indicating smaller but frequent purchases.

* 📌 **Revenue Distribution:** A small number of high-value transactions contribute a large portion of total revenue.

---

## 📊 Visualizations

The project includes multiple visualizations such as:

* Sales trend line charts

---

## 🚀 Conclusion

This analysis highlights key factors that influence retail sales, including customer demographics and product demand. These insights can help businesses optimize inventory, improve marketing strategies, and increase overall profitability.
